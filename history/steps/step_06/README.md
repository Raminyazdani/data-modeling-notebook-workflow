# Unsupervised Learning Workflow: Clustering & Dimensionality Reduction

**Practical implementation of K-means clustering and PCA**

## Overview

This project demonstrates unsupervised learning techniques including clustering and dimensionality reduction applied to digit recognition.

## Features

✅ K-means clustering with elbow method  
✅ Semi-supervised learning experiments  
✅ PCA dimensionality reduction  
✅ Performance evaluation across techniques  
🚧 Enhanced visualizations and documentation

## Setup

```bash
pip install -r requirements.txt
jupyter notebook unsupervised_learning_analysis.ipynb
```

## Tech Stack

- Python 3.x
- Jupyter Notebook
- scikit-learn
- NumPy, Matplotlib
