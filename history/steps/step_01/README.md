# Unsupervised Learning Workflow

**Exploring clustering and dimensionality reduction techniques**

## Overview

This project explores unsupervised learning methods for pattern discovery and data preprocessing in machine learning workflows.

## Goals

- Implement K-means clustering for pattern discovery
- Apply dimensionality reduction with PCA
- Explore semi-supervised learning approaches
- Build practical ML analysis skills

## Tech Stack

- Python 3.x
- Jupyter Notebook
- NumPy, Matplotlib, scikit-learn
